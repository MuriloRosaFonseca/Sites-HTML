Passos para Hospedar um Projeto no GitHub Pages

  1. Crie um repositorio no GitHub com o mesmo nome do seu projeto antigo
  2. No terminal, inicialize o repositorio local com Git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit com git commit - m "first commit".
  5. Conecte seu repositorio remoto com: git remotre add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main.
  7. No fitHub, vá em Settings > Pages
  8. Em "Branch", selecione maine a pasta / (root) . clique Save.
